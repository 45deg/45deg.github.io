CJK Markdown Fixer

通常のMarkdownレンダラーでは、強調記号の内側が「。」や「）」などの約物で、外側に日本語等のCJK文字が続くと、** を強調として認識できないことがあります。このツールは問題になる境界へ空白相当の文字を挿入し、テキストを修正します。処理はブラウザ内で完結し、入力したファイルや文章が外部へ送信されることはありません。 参考: tats-u/markdown-cjk-friendly
